API

Customer

Source code xxxx

Customer private informations

  • path: /customer/account
  • method: GET
  • is secured
  • Produces: CustomerPojoResponse

Customer public informations

  • path: /customer/permalink/{permalink}
  • method: GET
  • is not secured
  • Produces: CustomerPojoResponse

Customer create

  • path: /customer/
  • method: POST
  • is not secured
  • Consumes: CustomerPojo
  • 		{
			"login": "xxxx",
    		"title": "xxxx",
    		"firstname": "xxxx",
    		"lastname": "xxxx",
    		"email": "xxxx",
    		"password": "xxxx",
    		"defaultLocale": "xxxx",
    		"guest": "xxxx"
		}
  • Produces: void

Customer update

  • path: /customer/account
  • method: PUT
  • is secured
  • Consumes: CustomerPojo
  • 		{
			"login": "xxxx",
    		"title": "xxxx",
    		"firstname": "xxxx",
    		"lastname": "xxxx",
    		"email": "xxxx",
    		"password": "xxxx",
    		"defaultLocale": "xxxx",
    		"guest": "xxxx"
		}
  • Produces: void

Customer wishlist - product list

  • path: /customer/wishlist
  • method: GET
  • is secured
  • Produces: WishlistPojoResponse

Customer add to wishlist

  • path: /customer/wishlist
  • method: PUT
  • is secured
  • Consumes: WishlistPojoRequest
  • 		{
			"marketAreaCode": "xxxx",
			"catalogCategoryCode": "xxxx",
			"productSkuCode": "xxxx"
		}
  • Produces: AddToWishlistPojoResponse

Cart - details

  • path: /cart/
  • method: GET
  • is not secured
  • Consumes:
  • 		{
			"marketAreaCode": "xxxx"
		}
  • Produces: CommonCartPojoResponse

Cart - add item to cart

You need to call the "cart details" before, to get a cartId.

  • path: /cart/items/
  • method: POST
  • is not secured
  • Consumes:
  • 		{
			"marketAreaCode": "xxxx"
			"cartId": "xxxx"
			"productSkuCode": "xxxx"
			"catalogCategoryCode": "xxxx"
			"quantity": 1
		}
  • Produces: AddToCartPojoResponse

Cart - update item to cart

  • path: /cart/items/
  • method: PUT
  • is not secured
  • Consumes:
  • 		{
			"cartId": "xxxx",
			"productSkuCode": "xxxx",
			"quantity": 1
		}
  • Produces: CommonCartPojoResponse

Cart - update item to cart

  • path: /cart/items/
  • method: DELETE
  • is not secured
  • Consumes:
  • 		{
			"cartId": "xxxx",
			"productSkuCode": "xxxx"
		}
  • Produces: CommonCartPojoResponse
