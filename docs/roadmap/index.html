Roadmap

Improve the Rules Engine

  •  UI to save new rule on the "Business Backoffice".
  •  Rule engine to impact catalog product prices.
  •  Rule engine to impact shipping mode prices.
  •  Rule engine by gift code.

Refactoring the Workflow Engine

  •  Workflow engine to manage the "processing order" step.

Integration of the Dynamic Form Engine

  •  UI to define fields order and rules for each form in the website, like customer registration, newsletter, etc.
  •  Dynamic form engine to load and override the defined form validation rules.
  •  REST service to expose the rules.
